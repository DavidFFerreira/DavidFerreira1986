Configuração de Servidor Debian com Plex

Um guia completo para configurar o Plex no Debian a partir do Windows.

Requisitos Configurar Debian Instalar Plex Configurar Discos Permissões Aceder ao Plex

Requisitos

  • Uma máquina virtual ou física com Debian 11 ou 12.
  • Ligação à internet.
  • Discos disponíveis para armazenamento (NAS, USB ou internos).
  • Permissões administrativas para configurar o servidor.

Passo 1: Configurar Debian

Para começar, faça o download do Debian ISO e instale-o numa máquina virtual ou hardware físico.

Script Automático para Configuração Inicial

#!/bin/bash
# Atualizar sistema
apt update && apt upgrade -y

# Instalar utilitários básicos
apt install -y sudo vim curl wget git ufw

# Criar utilizador com permissões
read -p "Nome do novo utilizador: " USERNAME
adduser $USERNAME
usermod -aG sudo $USERNAME

# Configurar UFW
ufw allow OpenSSH
ufw enable

Guarde o script como setup-debian.sh e execute-o com:

sudo bash setup-debian.sh

Passo 2: Instalar Plex

Script para Instalação do Plex

#!/bin/bash
# Adicionar repositório Plex
echo "deb [signed-by=/usr/share/keyrings/plexmediaserver-archive-keyring.gpg] https://downloads.plex.tv/repo/deb public main" \
    | tee /etc/apt/sources.list.d/plexmediaserver.list

# Adicionar chave GPG
curl https://downloads.plex.tv/plex-keys/PlexSign.key \
    | gpg --dearmor -o /usr/share/keyrings/plexmediaserver-archive-keyring.gpg

# Instalar Plex
apt update
apt install -y plexmediaserver

Guarde o script como install-plex.sh e execute-o:

sudo bash install-plex.sh

Passo 3: Configurar Discos

NAS na Rede

sudo apt install -y cifs-utils
sudo mkdir /mnt/nas
sudo mount -t cifs //IP_DO_NAS/partilha /mnt/nas -o username=USER,password=PASS

Disco USB

sudo mkdir /mnt/usb
sudo mount /dev/sdX1 /mnt/usb

Disco Interno

sudo mkfs.ext4 /dev/sdX
sudo mkdir /mnt/interno
sudo mount /dev/sdX /mnt/interno

Passo 4: Configurar Permissões

sudo usermod -aG sudo,plugdev plex
sudo chown -R plex:plex /mnt/nas
sudo chown -R plex:plex /mnt/usb
sudo chown -R plex:plex /mnt/interno

Passo 5: Aceder ao Plex

Abra o navegador e aceda ao Plex em http://IP_DO_SERVIDOR:32400/web.

Inicie sessão e configure as bibliotecas para os discos montados.

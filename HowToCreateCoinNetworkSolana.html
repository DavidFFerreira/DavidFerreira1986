Manual de Criação de Token na Solana

Introdução

Bem-vindo ao guia completo para criar tokens na rede Solana. Este manual é adaptado para usuários de sistemas Windows, Linux e MacOS, e inclui um passo a passo detalhado para criar tokens SPL, bem como formas de automatizar o processo.

Pré-requisitos

  • Carteira Solana configurada (ex.: Phantom).
  • Instalação do Solana CLI e SPL Token CLI.
  • Tokens SOL na carteira para pagar taxas.

Instalação do Solana CLI

Para Windows

  1. Baixe o instalador do Solana CLI na página oficial: Clique aqui.
  2. Abra o PowerShell como administrador.
  3. Execute o comando fornecido no site oficial, por exemplo: 
    iwr -useb https://release.solana.com/v1.16.1/solana-install-init.ps1 | iex
  4. Reinicie o terminal para ativar o Solana CLI.

Para Linux/MacOS

  1. Abra o terminal.
  2. Execute o comando abaixo: 
    sh -c "$(curl -sSfL https://release.solana.com/v1.16.1/install)"
  3. Após a instalação, configure o PATH para o Solana CLI: 
    export PATH="/home/usuario/.local/share/solana/install/active_release/bin:$PATH"

Verifique se o Solana CLI foi instalado corretamente executando:

solana --version

Passos para Criar o Token

  1. Configure a rede Solana (Mainnet ou Devnet) com o comando: 
    solana config set --url https://api.devnet.solana.com
  2. Crie o token com: 
    spl-token create-token
  3. Crie uma conta para o token: 
    spl-token create-account <TOKEN_ADDRESS>
  4. Emita tokens para a conta: 
    spl-token mint <TOKEN_ADDRESS> <AMOUNT> <RECIPIENT_ADDRESS>

Instruções para Windows

Para usuários de Windows, o processo pode ser simplificado utilizando o terminal PowerShell. Certifique-se de adicionar as ferramentas CLI ao PATH durante a instalação.

  1. Abra o PowerShell.
  2. Execute: 
    spl-token create-token
  3. Continue com os mesmos comandos usados para Linux/MacOS, mas adaptados ao PowerShell.

Automatização com Scripts

Para facilitar o processo, você pode criar um script em Bash (Linux/MacOS) ou em PowerShell (Windows). Consulte exemplos completos no manual anterior ou peça ajuda adicional para personalização.

Conclusão

Com este guia, você agora tem o conhecimento necessário para criar e gerenciar seus próprios tokens na rede Solana, independentemente do sistema operacional que estiver usando. Aproveite e bons projetos!

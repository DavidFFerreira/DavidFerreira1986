3. Estrutura Mensal (4 Semanas)

Treinas 5 dias/semana (Seg→Sex). Tabata curto no fim de cada sessão. Supino & Remada aparecem todos os dias em baixa/alta intensidade alternada para reforço motor. Agachamento aparece nos dias de Pernas + em variantes no Tabata.

Semana Foco de Carga Volume Intensidade Notas
1 (Base Técnica) Aprender/ajustar técnica; cadências controladas. Médio RPE 6-7 Vídeo-teca/filmar execuções-chave.
2 (Progressão Volume) +1 série em principais; +reps onde qualidade mantida. Médio-Alto RPE 7-8 Tabata escala Nível 1→2.
3 (Progressão Intensidade) Aumentar carga 2,5–5% ou subir RPE. Médio RPE 8 Tabata completo se possível.
4 (Consolidação + Testes) Descarga leve 1ª metade, depois pequenos testes técnicos. Baixo-Médio RPE 6→9 (em teste) Reavaliar métricas corporais.

Intercalação de Exercícios (Semanas 1&3 vs 2&4)

Alterna os exercícios principais por grupo muscular para variar estímulo, reduzir sobrecargas repetitivas e manter motivação. Usa A para Semanas 1 e 3 e B para Semanas 2 e 4 .

Dia Bloco A (Sem.1/3) Bloco B (Sem.2/4)
Seg — Ombros Press Militar / Elevação Lateral / Face Pull / Rear Delt Fly / Arnold Press Arnold Press / Elevação Frontal / Shrug Barra / Face Pull / Cuban Press
Ter — Peito Supino Barra / Supino Inclinado DB / Crucifixo / Flexões / Peck Deck Supino Inclinado Barra / Supino Declinado DB / Crossover Cabo / Flexões Pés Altos / Chest Press
Qua — Braços Curl Barra / Hammer Curl / Tríceps Testa / Tríceps Corda / Kickback + extras Curl Alternado / Curl Scott / Tríceps Banco / Pushdown Barra / Extensão Overhead DB + extras
Qui — Dorsal Puxada Frente / Remada Curvada / Pull-over / Remada Baixa / Face Pull Remada Unilateral / Barra Fixa Assistida / Remada Máquina / Pulldown Neutro / Pull-over
Sex — Pernas Back Squat / Leg Press / Lunge / RDL / Ext+Flex Front Squat / Passada Step / Romanian DB / Hip Thrust / Hack Press ou Sissy

Nos dias em que um exercício base (Supino, Remada, Agachamento) não está pesado no bloco, executa 2 séries técnicas leves para reforço motor.
